# 🤖 AI Наставник по инженерии

Локальный AI-агент для обучения робототехнике, компьютерному зрению, тестированию и программированию. Работает полностью на вашем ПК с голосовым интерфейсом.

## ✨ Возможности

- 🧠 **Локальная LLM** (Qwen2.5-7B через Ollama)
- 📚 **RAG** - поиск по собственной базе знаний
- 🎤 **Голосовой ввод** (Vosk, русский + английский)
- 🔊 **Голосовой вывод** (Silero TTS)
- 🔍 **Поиск в интернете** (DuckDuckGo)
- 💻 **Выполнение Python кода** (с проверкой безопасности)
- 🌐 **Веб-интерфейс** (Gradio)

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt

### 2. Скачать модель Vosk
python download_vosk_model.py

### 3. Запустить Ollama с моделью
ollama pull qwen2.5:7b-instruct

### 4. Запустить веб-интерфейс
python src/web_ui.py

Открыть браузер: http://127.0.0.1:7860
